## Підсумок

Усі 6 функціональних кейсів — **PASS**. Ключова теза застосунку підтверджена
наочним заміром: WebGL тримає 120 fps на 50 000 точок, DOM просідає до 40 fps
маючи лише 2 000. Консоль чиста протягом усієї сесії. До власне застосунку
критичних багів не знайдено; знайдені проблеми стосувалися деплою (усунені під
час сесії) та документації.

## Розгортання (усунено під час сесії)

Спочатку демо віддавало 404. Діагностика й виправлення:

1. **GitHub Pages не увімкнено** → крок `deploy` падав з `Failed to create
   deployment (status: 404)`. Увімкнено Source: GitHub Actions.
2. **Секрет `VITE_MAPBOX_TOKEN` відсутній** → білд вбудовував порожній токен,
   застосунок показував fallback `no-token`. Секрет додано.
3. **Re-run failed jobs перевикористав старий артефакт** (білд без токена).
   Повний Re-run all jobs пересобрав із токеном (36s замість 16s).

Після цього карта завантажилась. Fallback-екран `no-token`, до речі,
відпрацював бездоганно — чистий меседж, жодного крешу (graceful degradation).

## Заміряні метрики (стрес-орбіта, 50 000 synthetic)

| Гілка | Точок | FPS | Worst frame | Data→visible / create |
|-------|-------|-----|-------------|------------------------|
| **WebGL** | 50 000 | **120 fps** | 9 ms | 176 ms data→visible |
| **DOM** | 2 000 (кап) | **40 fps** | 33 ms | 74 ms create |

25× більше точок при 3× вищому FPS. Idle-рендер 25k WebGL: 140–158 ms data→visible.

## Знахідки та спостереження (за спаданням важливості)

1. **Документація — застарілий URL.** README та BLOG посилаються на
   `virakachurovska.github.io/vue-mapbox-perf`; реальний деплой —
   `bambooplanedev.github.io/vue-mapbox-perf`. Треба поправити.
2. **Телеметрія Mapbox 503.** `events.mapbox.com/events/v2` (POST) стабільно
   віддає 503. Це аналітика Mapbox, не впливає на рендер; суто спостереження.
3. **Контраст тексту попапа.** Текст попапа (`point #id`, координати) —
   світло-сірий на білому, контраст низькуватий. Варто перевірити (можливо,
   лишок анімації появи).
4. **Латентність першого рендера.** На GitHub Pages + Mapbox дефолтний «геро»-
   стан домальовується за ~13–15 с; не баг, але сприймається як «порожня карта»,
   якщо дивитись у перші секунди. Можливий скелетон/лоадер для точок.
